# Pixel Chat

  ## Команды
  
    Запуск в режиме serve

    ```
    npm run start

    ```
    Сборка

    ```
    npm run build

    ```

    Локальная раздача статики через Express

    ```
    npm run serve

    ```

### Описание


    * /#login - по умолчанию
    * /#register
    * /#messanger
    * /#500
    * /#404
