# Pixel Chat

Мессенджер реализованный c использованием самописной библиотеки по отрисовки интерфейсов.

## Структура библиотеки

## Команды

    Запуск в режиме разработки

    ```
    npm run start

    ```

    Сборка

    ```
    npm run build

    ```

    Локальная раздача статики через Express

    ```
    npm run serve

    ```

    Тесты

    ```
    npm run test

    ```

### Страницы

    * /#login
    * /#register
    * /#messanger
    * /#error | 500
    * /#wrong | 404
